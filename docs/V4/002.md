# 运营端简介
!!! note ""
    高度集成的WuKongBox OS，全面提供空间项目（商公、联办、园区）日常工作中的 **运营**、**服务** 及 **物联智能硬件统一管理** 服务

***

<font color=#000 size=2>
    悟空宝盒运营端分为 **<u>公司视图</u>** 和 **<u>项目视图</u>** 两大功能分类。分别面向公司管理团队，实际项目运营团队，构成如下：

###公司视图
在公司视图下可以管理多个项目的运营

- <font color=#3F51B5> [报表分析](080.md)  </font>
